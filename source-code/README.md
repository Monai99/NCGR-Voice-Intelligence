# NCGR Smart Meeting Minutes — الأكواد المصدرية

نظام محاضر الاجتماعات الذكي — المركز الوطني للحوكمة (NCGR)

## محتويات الحزمة

| الملف | الوصف |
|---|---|
| `NCGR Meeting Minutes.dc.html` | **الكود المصدري الرئيسي** — واجهة التطبيق كاملة (الشاشات، المنطق، البيانات) |
| `support.js` | ملف التشغيل المساند لصيغة Design Component (مطلوب بجانب الملف المصدري) |
| `index.html` | **النسخة النهائية المستقلة** — ملف واحد مدموج يعمل بأي متصفح دون إنترنت |
| `assets/` | الشعارات والصور (شعار NCGR الأبيض والملوّن، إلخ) |

## طريقة التشغيل

- **الأسهل:** افتح `index.html` مباشرة بأي متصفح (نقرتان) — لا يحتاج خادمًا ولا إنترنت.
- **للتطوير:** ضع `NCGR Meeting Minutes.dc.html` و `support.js` ومجلد `assets/` في نفس المجلد على خادم محلي (مثل `python -m http.server`) ثم افتح الملف.

## الميزات

- تسجيل صوتي مع تفريغ نصي مباشر (Web Speech API — يتطلب متصفح Chrome وإذن الميكروفون)
- التقاط عنوان الاجتماع تلقائيًا عند نطقه («عنوان الاجتماع …»)
- توليد محضر منظّم: النقاط الرئيسية، القرارات، البنود المتفق عليها مع المسؤول
- سجل اجتماعات قابل للبحث مع تعديل العناوين والحذف (فردي/الكل)
- تحميل المحضر بقالب NCGR الرسمي
- واجهة عربية RTL بهوية NCGR (كحلي/فيروزي)

## ملاحظات تقنية

- توليد المحضر بالذكاء الاصطناعي (`window.claude.complete`) يعمل داخل بيئة Claude فقط؛ خارجها يُعرض التفريغ الملتقَط كنقاط بديلة.
- التفريغ الصوتي يعمل في المتصفحات الداعمة لـ Web Speech API (Chrome / Edge).
